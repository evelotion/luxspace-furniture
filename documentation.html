1. Gambaran Umum

3. Fitur Unggulan

LuxeBot AI

Rekomendasi furniture cerdas berbasis database produk secara real-time menggunakan Gemini AI.

Role-Based Access

Sistem keamanan yang memisahkan hak akses Admin dan Customer via Middleware.

Cinematic UI

Tampilan modern dengan Glassmorphism dan grid desain layaknya galeri seni.

4. Cara Instalasi

1. Install seluruh dependency:

npm install

2. Konfigurasi file .env dengan kredensial database lo:

DB_USER=postgres
DB_PASSWORD=yourpassword
DB_NAME=luxspace_db
GEMINI_API_KEY=your_api_key

3. Jalankan server:

npm start
